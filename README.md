# Climate-analysis-Data-science-
Automated climate data analysis tool using NASA GISTEMP records, featuring Scikit-Learn predictive modeling and high-resolution trend visualizations.
# 🌍 Climate Change Analysis & Prediction

## 📌 Overview
This project uses **NASA GISTEMP v4** data to analyze global temperature anomalies since 1880. It utilizes a **Linear Regression** machine learning model to forecast temperature trends through the mid-21st century.
